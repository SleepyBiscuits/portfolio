Milkitty

Milkitty was a interactive site chalslenge, where it involved its own custom vectors and animations.

Milkitty Prototype and Wireframing

The behind the scenes on conceptualising Milkitty and its design.

Front-end Development project

The website would make use of web elements and multimedia features to showcase a (fictional) milk tea café namely Milkitty, which was a concept challenge for my friend (Veronica) to design. Within 24 weeks or semesters worth of work, I only manage to design a simple but fairly animated website though it lacks the interactivity I originally wanted.

Main Priority

  • Create a single page site that conveys an experience at Milkitty café.
  • Drafts, Prototypes, and flow of the content
  • Adding animations and visuals
  • Finally, a working animated webpage

Design and Development process

credits to Veronica Faith Guillermo for concept and design

The Milkitty prototype and wireframing process started with sketching several ideas before deciding on a final draft for the flow of the website.

Afterwards, I used Adobe Illustrator to create the various vectors and shapes for the visuals I needed (e.g. the bobas, the floating cat, and the drinks). Finally prototyping everything onto figma.

Draft Website Flow

Initial Prototype with vectors


Sample Vectors

For the floating cat portion, I wanted to make the cat feel alive chilling on top of the floating boba. I originally wanted to animate the cat using javascript and a bit of creative css magic, but ultimately after some attempts, I decided to animate it into a Lottie animation through After Effects.



Cat Animation Attempts

Tweening into a Lottie Animation

Finally, putting together the website using some html, css and javascript. Making use animated libraries like GSAP as well. It went through some iterations and experiments since this was my first attempt at a highly visual website. Overall I'm satisfied that I achieved my main goals for the project while juggling a bunch of university projects on the side at the same time.
I might get back on this soon, so for now enjoy the milkitty experience!
